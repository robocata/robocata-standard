# Changelog

All notable changes to the RoboCata Standard will be documented in this file.

## [1.0] - March 2026

### Initial Release

- 267 fields across 24 specification sections
- 6 required fields: `robot_code`, `info_name`, `manufacturer_text`, `type`, `domain`, `category`
- 72 coded option fields with predefined valid codes
- 15 type codes, 29 category codes, 3 domain codes
- Robot Code format: RBC-TT-CC-XXXXXXX
- JSON Schema (Draft 2020-12)
- Machine-readable codelist files
- Conformance requirements
